# 2、使用方法

> 📌 <font color="#FF6699">**扩展属性**</font>
>
> `WatermarkTextBox` 提供了一些扩展属性：
>
> * **WaterText**：获取或设置水印文字的文本；
> * **WaterMarkFont**：获取或设置水印文字的字体；
> * **WaterMarkLocation**：获取或设置水印文字左上角相对于文本框左上角的坐标；
>
