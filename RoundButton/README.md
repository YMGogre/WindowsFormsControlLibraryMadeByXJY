> 📌 <font color="#FF6699">**注意**</font>
>
> 由于 Windows 11 对系统 UI 视觉设计的更改，Win11 默认的按钮样式已经更改为圆角按钮了。而 Winform Button 控件会根据操作系统的主题和样式进行渲染，所以 Winform 的按钮控件在 Win11 上已经默认显示为圆角了。
